# 🏔️ Kumaon Stays

A MERN Stack Hotel Booking Web Application.

## 🚀 Tech Stack

- React + Vite
- Node.js
- Express.js
- MongoDB
- JWT Authentication

## 📂 Project Structure

```
Kumaon-Stays/
│
├── frontend/
├── backend/
└── README.md
```

## Features

- User Registration
- User Login
- Hotel Listings
- Hotel Details
- Add Hotels
- Delete Hotels
- JWT Authentication
- Responsive UI

## Run Project

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
